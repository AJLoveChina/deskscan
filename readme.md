### deskscan
> 一个可以捕捉桌面二维码并解析的命令行工具

### Usage
```text
npm install -g deskscan
```
Then
```text
deskscan
```
For help information
```text
deskscan -info
```

### License
MIT

### 工作原理
* 捕捉桌面并生成图片（多个显示器会全部捕捉）
* 查找图片中的二维码
* 解析二维码并输出结果到控制台

### 适用平台
- [x] windows
- [x] Mac
